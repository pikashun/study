# 固定格式
    <div class="container">//container表示根据屏幕尺寸调整固定宽度，container-fluid表示始终为屏幕尺寸的100%;
        <div class="row">
            //这里写<div class="col"><div>
            //col-xl-2;此类格式固定12列xl表示低于此尺寸时，每一行只展示一列,2代表独占两列，只用col-无论屏幕尺寸多少，都会是12列
            //这里写=若class="col"则会创建等宽列，写多少个都会自动均分父级宽度
        <div>
    </div>
